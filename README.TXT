IntelligentGridNavigator - Autonomous Robot Pathfinder

An autonomous, grid-based intelligent system built in Java for AP Computer Science A. The system uses recursive algorithms to generate complex mazes and Breadth-First Search (BFS) pathfinding to navigate an agent toward a designated rescue target.

Technical Specifications
- Core Structure: 2D Array Grid (GridCell[][])
- Algorithms: Recursive Backtracking (Maze Generation), Breadth-First Search (Pathfinding)
- Architecture: Object-Oriented Design (Min. 8 classes with encapsulation and inheritance)
- GUI: Java Swing interface for real-time visualization

How to Run
1. Ensure Java JDK 8 or higher is installed.
2. Compile all files: javac *.java
3. Run the application: java MainApp
